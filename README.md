# Cash App UI Clone

A modern Cash App-style interface built with React, Vite, and Tailwind CSS.

## Features

- ✅ **CRUD Operations**: Create, Read, Update, and Delete transactions
- ✅ **Real-time Search**: Filter transactions by name, description, or amount
- ✅ **Online Avatars**: Dynamic profile pictures using UI Avatars API
- ✅ **Responsive Design**: Mobile-first design matching Cash App aesthetics
- ✅ **Modern UI**: Dark theme with green accents

## Tech Stack

- React 19
- Vite 7
- Tailwind CSS 4
- PostCSS

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Features in Detail

### Transaction Management
- Click the green "+" button to add new transactions
- Hover over transactions to reveal edit and delete buttons
- Search bar filters transactions in real-time

### UI Components
- Activity feed with transaction history
- Stories row with user avatars
- "Add money" section
- Bottom navigation bar
